## Add new feature to the project guide line step by step

### 1. Create a new branch from master branch

    ```bash
    git checkout master
    git pull origin master
    git checkout -b feature/feature-name
    ```

### 2. Create a new component

    ```bash
    cd src/components
    mkdir pageName
    cd pageName
    touch index.js
    ```
